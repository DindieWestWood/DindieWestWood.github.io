Byblos

Description

Byblos is a puzzle game for mobile devices (Android). The purpose of the game is simple, the player has to move spheres on corresponding tiles (identified with symbols), in a limited amount of time. The player can only move up, right, down or left. If there is several spheres on the board they are moving altogether. Throughout the game, some obstacles and special tiles will appear.

This project is currently in development since a year. We are a small team: 2 developers (myself included) and a composer, and we are developing on our spare time. We expect to release the game during the upcoming year.
